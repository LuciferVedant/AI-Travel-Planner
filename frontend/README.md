# Trao AI - Frontend

The user interface for the Trao AI Travel Planner.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: Context API (AuthContext)

## Features
- **Modern UI**: Premium design using glassmorphism and responsiveness.
- **Dynamic Dashboards**: User-specific views for itineraries.
- **Multi-step Forms**: Seamless data collection for trip generation.
- **Itinerary Management**: Interactive views for viewing and modifying itineraries.

## Scripts
- `npm run dev`: Starts the development server.
- `npm run build`: Creates a production build.
- `npm start`: Runs the production server.

## Design
The project uses a custom design system defined in `src/app/globals.css`, focusing on high-end aesthetics with gradients and glass-like components.
